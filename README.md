# 🌑 MidnightTrioTracker (v12.0.1)

![WoW Version](https://img.shields.io/badge/WoW-12.0.1%20Midnight-blueviolet)
![License](https://img.shields.io/badge/License-MIT-green)
![Author](https://img.shields.io/badge/Author-Cosotozzo-orange)

**MidnightTrioTracker** is a lightweight, high-performance addon designed for World of Warcraft: Midnight. It provides a streamlined way to track your weekly progress across the three core activities of the current endgame.

---

## ✨ Key Features

* **Trio Tracking:** Real-time tracking for Raids, Mythic+, and the new Void Delves/Expeditions.
* **Performance Optimized:** Built with a "Zero-Waste" philosophy. No constant `OnUpdate` polling—everything is event-driven to preserve your FPS.
* **Modern UI:** Built using the Dragonflight/Midnight UI templates for a native look and feel.
* **Auto-Update Ready:** Fully compatible with WowUp, CurseForge, and Wago.io.

---

## 🛠 Slash Commands

Type these commands in-game to manage the interface:

| Command | Action |
| :--- | :--- |
| `/mtt` | Toggle the main tracker window |
| `/mtt reset` | Reset the window position to the center |
| `/mtt config` | Open the options panel (Coming Soon) |

## 📦 Installation

### Via Addon Manager (Recommended)
1. Open your preferred manager (WowUp, CurseForge, etc.).
2. Search for `MidnightTrioTracker`.
3. Click **Install**.

### Manual Installation
1. Download the latest [Release](https://github.com/Cosotozzo/MidnightTrioTracker/releases).
2. Extract the folder into `_retail_/Interface/AddOns/`.
3. Restart World of Warcraft.
